Pseudo-Elements

::before & ::after
Insert content before or after an element's content. Great for icons, decorations, or small labels.
Decorated paragraph using ::before and ::after.
<p class="demo before-after">Text</p>
.demo.before-after::before{ content: " * "; }
.demo.before-after::after { content: " # "; }
::first-letter
Style the first letter of a block — used for drop-caps or emphasis.
Once upon a time, this paragraph uses ::first-letter to create a drop-cap style.
<p class="demo first-letter">Once upon...</p>
.demo.first-letter::first-letter{ font-size:28px; color:#b91c1c }
::first-line
Style only the very first line of text (depends on line width).
This line is styled with ::first-line. Note: it affects exactly the first rendered line, so resizing the window changes the effect.
<p class="demo first-line">Lorem ipsum...</p>
.demo.first-line::first-line{ font-weight:700; color:#0b5aa6 }
Notes
Pseudo-elements target parts of elements. Use double-colon `::` (modern) though single-colon works in older browsers for some.
::before, ::after, ::first-letter, ::first-line

User Interaction

:hover
Apply style when the mouse is over an element.
.btn-basic:hover { transform: translateY(-3px); box-shadow: 0 6px 18px rgba(...); }
:active
Style when the element is being clicked (pressed).
.btn-basic:active { filter: brightness(.92); transform: translateY(0); }
:focus
When an element receives focus (keyboard typing, tabbing). Useful for accessibility.
input:focus { box-shadow: 0 0 0 4px rgba(37,99,235,0.09); outline: none; }
:checked
Applies to checked radio buttons and checkboxes.
      
input[type="checkbox"]:checked + label { /* or input:checked { } */ }

Link States

Structural Selectors (Position)

:first-child / :last-child
Select elements based on their position among siblings.
  1. Item 1 (first-child)
  2. Item 2
  3. Item 3
  4. Item 4 (last-child)
ol li:first-child { font-weight:700; }
ol li:last-child  { font-style:italic; }
:nth-child() / :nth-last-child()
Select the nth child (1-based). Supports formulas like 2n, 2n+1, odd, even.

  • 2
  • 3
  • 4

CSS highlights are applied via stylesheet (see snippet below).

ul li:nth-child(odd) { background:#f8fafc; }
ul li:nth-child(3)   { font-weight:700; }
:nth-last-child()
Select counting from the end (last child is 1).
  1. First
  2. Second
  3. Third
  4. Fourth
ol li:nth-last-child(2) { background:#fff; border-left:3px solid var(--accent); }
Quick tip
Use :nth-child(an+b) for repeating patterns (e.g., 2n for even rows).
li:nth-child(2n) { background: #f3f4f6 }

Form State Selectors

:valid / :invalid
Validate inputs with HTML attributes (type, pattern, required) and style them accordingly.
input:valid   { border-color: green }
input:invalid { border-color: red }
:disabled / :enabled
Style disabled controls (grayed out) and enabled controls differently.
  
button:disabled { opacity:.5; cursor:not-allowed }
